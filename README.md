# dominos-pizza-sales-analysis
An end-to-end sales data analysis project using Python, Pandas, NumPy, and Matplotlib to uncover business insights, customer trends, and sales performance through interactive visualizations.

## Project Overview

This project analyzes Domino's Pizza sales data using Python to uncover customer purchasing patterns, sales trends, and product performance. The objective is to transform raw sales data into meaningful business insights that can help improve sales strategies, inventory planning, and customer satisfaction.

The project involves data cleaning, exploratory data analysis (EDA), and data visualization using Python libraries.

---

## Objectives

- Analyze overall sales performance.
- Identify the best-selling and least-selling pizzas.
- Determine peak sales periods.
- Analyze revenue by pizza category and size.
- Identify customer ordering trends.
- Generate business recommendations based on data insights.

## Dataset Information

The dataset contains historical Domino's Pizza sales transactions including:

- Order ID
- Order Date
- Order Time
- Pizza Name
- Pizza Category
- Pizza Size
- Quantity Ordered
- Unit Price
- Total Price

---

## Data Cleaning

The following preprocessing steps were performed:

- Checked for missing values.
- Removed duplicate records.
- Converted date columns into datetime format.
- Extracted month, day, and hour from order dates.
- Verified data types.
- Prepared the dataset for analysis.

## Key Insights

- Large-sized pizzas generated the highest revenue.
- Weekend sales were higher than weekdays.
- Evening hours experienced the maximum number of orders.
- Classic category pizzas contributed significantly to total sales.
- A small number of pizza varieties generated a major share of revenue.

---

## Business Recommendations

- Increase inventory during weekends.
- Promote low-selling pizzas through discounts.
- Introduce combo offers during non-peak hours.
- Optimize staffing during evening rush hours.
- Focus marketing campaigns on high-performing pizza categories.
